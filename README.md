# Roda Mirroring

Tool to mirror a screen region and share it in Google Meet without exposing the rest of the screen.

## Requirements
- Linux (tested on Linux Mint)
- Python 3
- PyQt5, mss, pillow

## Install dependencies
pip install PyQt5 mss pillow --break-system-packages

## Run
python3 screen_mirror.py

## Build AppImage
bash build_appimage.sh
